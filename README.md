# Snakes

Another console snake game. Created using Python 3 and curses library.

## Installation
Install Python 3 and then:

For Windows

```bash
git clone https://github.com/brebiv/Snakes.git
cd Snakes
pip install windows-curses
python snake.py
```
For linux
```bash
git clone https://github.com/brebiv/Snakes.git
cd Snakes
python3 snake.py
```

## Controls
Use arrows to control the snake.
